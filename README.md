# DataFirewallFlink

## 📌 Описание
DataFirewallFlink — это проект для обработки и фильтрации потоковых данных на базе Apache Flink.
Предназначен для анализа, валидации и маршрутизации входящих данных в реальном времени.

## 🛠 Технологии
- Java 17  
- Apache Flink  
- Maven  
- Git
